# 🃏 Campeonato de Truco

Aplicativo mobile desenvolvido para gerenciamento completo de campeonatos de truco, incluindo rankings, partidas e organização de torneios.

---

# ✨ Funcionalidades

- 🏆 Sistema de ranking
- 🎯 Criação de campeonatos
- ⚔️ Controle de partidas
- 👥 Gestão de equipes
- 📊 Estatísticas
- 🔄 Atualizações em tempo real
